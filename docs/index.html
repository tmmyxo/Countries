Terratorries

About the app

With this app you are able to browse countries and other territories of the world, see basic data about them, and view their locations on the world map. You can play a mini-game to memorize flags of countries you're interested in, and finally, you have a MapView that allows you to save locations by placing a point and giving it a name. You can also save your current location as a point and see a short or full address of the selected point in a pop-over view.

Technologies used

  • Swift
  • UIKit
  • WebKit
  • MapKit
  • CoreLocation
  • CoreData

Key Features

Expandable adress view

The view is presented whenever a pin on a map is selected. By default it displays short adress of the pin location but it can be expanded by tapping or dragging. You can copy adress by holding on it

Neumorphic Buttons

Practice tab greets user with beautiful neumorphic buttons, making the illusion of a 3D object just by using shadows. To extend button perception it has haptic feedback on touch and release.

Tab Selection Slider

To display currently selected tab, smooth slider has been implemented, it will nicely transforms into a search button once "Coordinates" tab is selected. The button itself has haptic feedback to recreate the feeling of a real button.

Dark Mode Support

To support dark mode, I've created a color scheme with complimenting colors for both light and dark modes. To implement it, I designed two variants of used assets and created dynamic UIColor pairs for elements and shadows. Every color will choose its appropriate value automatically whenever the system appearence changes.
